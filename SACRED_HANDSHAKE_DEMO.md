## 📊 Integration Patterns

### Pattern 1: Single Agent Activation

```
1. Copy agent prompt (short or full)
2. Paste into AI chat
3. AI assumes agent role
4. Interact with agent
```

### Pattern 2: Multi-Agent Coordination

```
1. Establish Grid context (handshake)
2. Activate primary agent (e.g., Mo)
3. Mo coordinates with other agents as needed
4. Unified response from Grid
```

### Pattern 3: External System Integration

```
1. Export prompts to JSON
2. Load into your application
3. Send appropriate prompt based on task
4. Process AI response
```
